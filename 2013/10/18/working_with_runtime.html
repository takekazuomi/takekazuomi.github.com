Working with Runtime

Azure上にアプリケーションを実行しよとするとアプリケーションをがランタイムを要求することがあります。ランタイムはあまり変更されないので、パッケージ（cspkg）に入れて毎回アップロードするのは時間がかかり無駄です。また、エミュレータ上では開発用にローカルにインストールされたものを使ったほうがアプリケーションの実行開始にかかる時間が少なくなります。

Windows Azure Plugin for Eclipse with Java では、JDKをカスタマイズものをruntimeとして使う機能が用意されているので、ここでは、Play Framework 1.2とjreをruntimeとして設定する方法を紹介します。

runtimeの準備
